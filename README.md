### python中的数据类型：

Data stractures

* list - sequences of mutable values   //列表可以改变[]
* tuple - sequence of immutable values //元组不可改变()
* set - collection of unique values    //集合不可以重复()
* dict - collection of key-value pairs  // 字典键值对{}
